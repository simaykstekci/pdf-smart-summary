📄 PDF Smart Summary

PDF Smart Summary, PDF dosyalarındaki metinleri otomatik olarak analiz eden ve özetleyen bir Python uygulamasıdır.
Bu proje, uzun PDF dokümanlarını hızlıca anlamayı ve önemli bilgileri öne çıkarmayı amaçlar.

🚀 Özellikler

📑 PDF dosyasından metin çıkarma

✂️ Uzun metinleri otomatik olarak özetleme

⚡ Hızlı ve kolay kullanım

🧠 Doğal dil işleme (NLP) teknikleri kullanımı

🛠 Kullanılan Teknolojiler

Python

PyPDF2 / pdfplumber (PDF okuma)

NLTK / spaCy / transformers (kullandığına göre sil veya bırak)

TextRank / AI tabanlı özetleme (varsa)

📦 Kurulum
git clone https://github.com/kullaniciadi/pdf-smart-summary.git
cd pdf-smart-summary
pip install -r requirements.txt

▶️ Kullanım
python main.py sample.pdf


Program, verilen PDF dosyasını analiz eder ve özet çıktısını ekrana yazdırır.
